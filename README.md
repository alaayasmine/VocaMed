# VocaMed
## 🌟 Overview

Vocamed is an AI-powered medical scribe system designed to help healthcare professionals generate structured medical reports in real time during patient consultations.

The system records conversations, transcribes audio using AI speech-to-text services, and automatically structures medical information into SOAP format.

---

## 🎯 Purpose

The main goal of Vocamed is to reduce the time spent on manual medical documentation by automating report generation while maintaining clinical structure and workflow efficiency.

---

## ⚙️ Features

### 👤 Patient Management
- Patient registration system
- Triage information updates
- Patient status tracking
- Database storage of medical encounters

### 🎙️ Real-Time Recording
- Browser-based live audio recording
- Consultation capture during visits

### 🤖 AI Integration
- Speech-to-text transcription using AI services
- Medical report generation using language models
- Structured SOAP output formatting

### 📊 Dashboard Interface
Supports three user roles:

- Reception staff – Patient registration
- Nurses – Triage data entry
- Doctors – Consultation recording and report generation
